# Autumn's Dotfiles

## Instructions

1. Clone this repository into your home directory
2. Install stow with `brew install stow`
2. `cd` into the `dotfiles` directory
3. run `stow .`

## Dotfiles Included

- [x] `.tmux/*`
- [x] `.tmux.conf`
- [x] `.aws/config`
- [x] `.gitconfig`
- [x] `.zprofile`
- [x] `.zshrc`
- [x] `zen.toml`
